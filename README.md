# DelishBite (Vite + React + Tailwind)

This is a minimal Vite + React + Tailwind project scaffold created in this workspace.

Quick start (PowerShell on Windows):

```powershell
cd "E:\Educational Tools\vite\DelishBite_restaurant"
npm install
npm run dev
```

Open the URL printed by Vite (usually http://localhost:5173) to see the app.

Notes:
- Tailwind is configured in `tailwind.config.cjs`.
- Entry point: `src/main.jsx` and `src/App.jsx`.
